  ```scss main.scss focus=4:11
      $media-padding: true,
      $media-margin: true,
  ```

  add important flag to utility
  ```scss main.scss focus=2:4
  @use "basket" as * with (
    $utilities: (
      important: true,
    ),
  );
  ```
  --- 

  create new utility class

  ```scss main.scss focus=2:15
  @use "basket" as * with (
    $utilities: (
      "my-util": (
        class: "uitl",
        state: hover,
        property: color,
        media: true,
        important: true,
        values: (
          ppp: #fff,
          ddd: #ddd,
          eee: #ccc
        ),
      )
    ),
  );
  ```
